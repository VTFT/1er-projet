Cool features

Here you can see what Boe Tea has to offer.

Find images source

You can find where your image came from by simply posting it and typing "bt!sauce" in the channel.

Block reposts

You're getting tired of seeing the same pictures posted every 3 messages ?
Good news, you can block it with Boe Tea.

Save your favorites

You saw something you like ? Save it in your favorites list simply by clicking on the heart reaction for SFW and drooling face for NSFW.

Display all images

Hey mobile users, we didn't forget you. Whenever there are 2 or more images in the same link, Boe Tea will display them fully one by one.

Block NSFW content

You want to keep your server family friendly ? You can block content that is tagged as explicit.

There is more

If you want to know all commands available, the full documentation is there for you.
